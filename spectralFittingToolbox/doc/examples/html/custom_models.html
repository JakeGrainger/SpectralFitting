Custom Models

The user can specify there own spectral density functions.
They can also provide analytical forms for the aliased spectral density and autocovariance if they wish.
To do this, set the appropriate property to the appropriate function handle.
These functions should also calculate derivatives and second derivatives in the same way as genJONSWAPsdf.
help spectralFitting.genJONSWAPsdf
genJONSWAPsdf function to evaluate a generalise JONSWAP spectra [sdf,d_sdf,d2_sdf] = genJONSWAPsdf(parameter,omega) computes the generalised JONSWAP spectral density with parameters defined by parameter = (alpha,omega_p,gamma,r) at angular frequencies given by omega. Also computes first and second derivatives, with first deriviatves stacked in columns (i.e. d_sdf(:,p) is the pth derivative). The second deriviative is similar, except that only the lower triangle is give (as the second deriviative matrix is symetric. The order given goes down eash column of the lower triangle in turn. See also bimodalSeaSdf.
