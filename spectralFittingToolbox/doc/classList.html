Classes and Methods

List of classes and their respective methods


Spectral Fitting Toolbox includes a few different classes used to perform the different features.


SpectralModel

This is the main class used by Spectral Fitting Toolbox. Objects of this class define spectral models. These models can be simulated from or fitted to data using the appropriate methods.

NonParaSpcEst

This class allows the user to easily calculate and plot the non-parametric estimates of a timeseries. The two subclasses Periodogram and BartlettPeriodogram should be used to construct the relevant estimate. But the plotting functions are shared between the two.

Methods

  • plot
  • plotdB
